Q. How do I decode audio?

A. From Java, see the SDK API android.media.MediaCodec.
From C or C++, see the corresponding NDK API <media/NdkMediaCodec.h>.

Q. Where is the documentation for the NDK version of the MediaCodec API?

A. There is no documentation currently; in the mean time see the sample code at native-codec.
